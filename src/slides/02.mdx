# What's new in Chakra UI v1?

- 🤠 100% TypeScript rewrite
- 🏭 `chakra` factory function
- 🌐 Theme-aware `sx` prop
- 🎨 Theming API & global styles
- 🧬 New components and hooks

See the v1 docs at https://next.chakra-ui.com/ for more detailed information. Be
sure to check out the "Upgrade to v1" doc if you're upgrading to v1 from an
existing v0 project.
